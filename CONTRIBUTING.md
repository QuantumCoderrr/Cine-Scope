# 🤝 Contributing to Cine-Scope

Thanks for considering contributing to **Cine-Scope** — a fully frontend movie explorer built with love using TMDB API.

---

## 📦 Local Setup

1. Fork the repo
2. Clone it:
   ```bash
   git clone https://github.com/QuantumCoderrr/Cine-Scope.git
   cd Cine-Scope```

3. Open index.html in your browser — no build needed.

## 🚀 Contribution Guidelines

✅ Keep commits focused: One feature or bugfix per PR <br>
✅ Use clear commit messages (e.g., fix: rating stars update properly) <br>

## 🛠️ What Can You Contribute?

Here are a few areas where you can help improve **Cine-Scope**:

- ✨ **Improve UI/UX** – Clean up layout, animations, responsiveness
- 🎛️ **Add new filters** – Runtime, language, certification, etc.
- 🧱 **Convert to PWA** – Make Cine-Scope installable as a web app
- 🔐 **Integrate Firebase or Supabase** – For auth, storage, or watchlist syncing
- ♿ **Enhance Accessibility** – Use semantic tags, ARIA roles, keyboard navigation
- ⚡ **Boost Performance** – Lazy load images, reduce render time, minify assets
